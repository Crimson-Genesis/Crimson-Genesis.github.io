A retro-styled web interface for browsing my research papers, notes, and linked documents.

The site includes:
- Local & remote Markdown / PDF viewing  
- Full-text search (title + markdown content)  
- Dark & light themes  
- URL-based document loading using `?src=`  
- Smooth internal scrolling with pixel-art UI  

Everything is static and fully client-side, making it ideal for GitHub Pages hosting.

This repository contains the full source code powering the site.
